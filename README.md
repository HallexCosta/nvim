## Requeriments
* NeoVim ^0.3.0

## Install Neovim
**Windows**
> **choco install neovim**

**Linux**
> **sudo add-apt-repository ppa:neovim-ppa/stable**
> **sudo apt-get update**
> **sudo apt-get install neovim**
